# Canva Asset Plan

This repository already has strong public-safe AIXYZ visuals, so the current
package reuses existing project assets instead of generating replacements.
This plan defines the Canva-ready direction for future refreshes.

## Hero Image

- Purpose: README hero, WordPress hero, and project landing context.
- Dimensions: 1600 x 900.
- Style direction: polished AI web studio, human-operated systems, public demo layer.
- Color direction: black, white, apple green, warm accent highlights.
- Typography direction: bold geometric headline, readable supporting labels.
- Canva prompt: Create a premium AI web studio hero for AIXYZ showing a public website, demo workflow, lead intake, and protected operations layer without exposing code or dashboards.
- Alternate Canva prompt: Design a confident public project hero for an AI website studio with layered workflow cards, clean screenshots, and apple-green accents.
- Alt text: AIXYZ AI web studio hero image.

## GitHub Banner

- Purpose: visual impact at the top of the GitHub README.
- Dimensions: 1600 x 500.
- Style direction: wide editorial banner with the AIXYZ name, public surface, protected engine, and receipts concept.
- Color direction: dark neutral base with apple-green signal lines and white type.
- Typography direction: concise title treatment with smaller supporting phrase.
- Canva prompt: Create a wide GitHub repository banner for AIXYZ, a protected public surface for an AI web studio, with workflow lines and public/private boundary cues.
- Alternate Canva prompt: Make a cinematic but readable AIXYZ project banner showing website offers, demos, intake, review, and receipts.
- Alt text: AIXYZ GitHub banner.

- Typography direction: large readable title, compact supporting copy.
- Canva prompt: Create a social preview card for AIXYZ AI Web Studio with public-safe demo and workflow cues, polished but not generic SaaS.
- Alternate Canva prompt: Design a social card for AIXYZ showing AI websites, lead capture, and owner review in a protected public project surface.
- Alt text: AIXYZ branded social card.
