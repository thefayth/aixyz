# Brand Style Notes

## Brand Feel

AIXYZ should feel bold, intelligent, human, strategic, and memorable. It should
not feel like a generic SaaS template.

## Visual Direction

- Dark technical base.
- Apple green accent instead of blinding neon.
- Graphic AI/web studio visual language.
- Public-safe screenshots and workflow diagrams.
- Clear ownership and protected-engine messaging.

## Existing Assets Used

- AIXYZ generated hero.
- Auntie AI social/brand assets.
- Production AIXYZ screenshots.
- Canva workflow graphic.

## Accessibility Notes

- Use descriptive alt text.
- Avoid relying on color alone.
- Keep public/private boundary language explicit.
