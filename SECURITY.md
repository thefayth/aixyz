# Security

## Reporting

Send security concerns through Faith Cheltenham's public contact channels at
[https://faithcheltenham.com](https://faithcheltenham.com).

## Public Boundary

This repository does not contain production source code, credentials, tokens,
cookies, private customer data, deployment infrastructure, payment data, or
private operational records.

Do not submit credentials, private customer material, account screenshots, or
security-sensitive details through public issues.

## Supported Surface

This is a public project and documentation surface. Private systems and
production operations are not supported through this repository.
